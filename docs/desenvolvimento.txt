Dependências
------------

*Atenção: Esta seção é destinada a usuários avançados do XMPPVOX
que desejam alterar seu código-fonte. Se você não sabe o que isto
significa, pode ignorar esta seção.*

Para utilizar o XMPPVOX é preciso que estejam instalados os seguintes
programas e bibliotecas:

    - DOSVOX / Papovox (testado com DOSVOX 4.2alfa, Papovox 3.3B)
    - Python 2.7 (testado com 2.7.2)
    - dnspython (versão incluída no diretório thirdparty)
    - SleekXMPP (versão incluída no diretório thirdparty)

Dependências para gerar executável do Windows:

    - pywin32 (testado com Build216)
    - PyInstaller (testado com versão de desenvolvimento trunk-1899)
    - UPX (opcional, testado com versão 3.08)

Dependências para usar scripts ".sh":

    - MinGW
